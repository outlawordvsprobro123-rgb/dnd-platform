# Правила React компонентов

- 'use client' — только при использовании useState, useEffect, useRef, event handlers
- Без 'use client' — компонент автоматически Server Component (предпочтительно)
- Konva компоненты — всегда 'use client', импорт динамический: `dynamic(() => import('react-konva'), { ssr: false })`
- Howler.js — всегда 'use client', инициализация только в useEffect (не при SSR)
- Loading состояния — обязательны для любого async действия (disabled кнопка + спиннер)
- Оптимистичные апдейты — применять для HP изменений и движения токенов
- Realtime подписки — в отдельных хуках `use{Module}Channel.ts`, не в компонентах напрямую
- Текст — всегда на русском, никаких английских лейблов в UI
- Tailwind классы — не использовать произвольные значения `[]` если есть стандартный класс
- Компонент > 200 строк — разбить на подкомпоненты
